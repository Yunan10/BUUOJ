# 注意！！！
数据包里要包含**请求行、请求头、空行和请求体**四个部分，构成一个标准数据包
即使你发了一个GET请求数据包，根本不需要请求体，那你还是得**把空行和请求体的空位留出来**
最后要**空两行！！！一行是空行，一行是请求体**
